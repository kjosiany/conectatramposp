🔗 ConectaTrampo SP

O ConectaTrampo SP é um site desenvolvido para divulgação gratuita de vagas de emprego na cidade de São Paulo e região.

O projeto tem como objetivo facilitar o acesso a oportunidades de trabalho por meio de uma interface simples e organizada, permitindo que usuários visualizem vagas disponíveis de forma rápida e acessível.

Este sistema foi desenvolvido utilizando tecnologias básicas da web, com foco em prática de desenvolvimento front-end e organização de conteúdo.

🚀 Tecnologias Utilizadas
HTML
CSS
JavaScript

📋 Funcionalidades
Exibição de vagas de emprego
Organização das oportunidades por páginas
Navegação simples entre conteúdos
Interface visual acessível
Estrutura base para um portal de divulgação de vagas
